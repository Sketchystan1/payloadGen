# payloadGen

Simple [payload Generator](https://sketchystan1.github.io/payloadGen/) for `i1..i5` hex lines.

## Scripts

- [hex2shark.ps1](https://github.com/Sketchystan1/payloadGen/blob/master/hex2shark.ps1) - Takes `iX=<b 0x...>` lines or `Hex Streams` pasted into the console and opens in [Wireshark](https://www.wireshark.org/download.html).

- [tshark_capture.ps1](https://github.com/Sketchystan1/payloadGen/blob/master/tshark_capture.ps1) - Captures auto detected browser traffic with [Wireshark](https://www.wireshark.org/download.html) and saves `.pcapng` and `.txt` report with `iX=<b 0x...>` lines and `Hex Streams`.
